## tech stack
- sveltekit for ui
- expressjs for backend
- pupetter for webscraping
- daisyui for fast and quick components
- tailwind (ofc)
- jwt
- mongodb

## how to setup locally

### running the server
```
cd server
npm i
npm run dev
```
### runnning the client
```
cd ui
npm i
npm run dev
```
